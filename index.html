TNGPC – Police Canteen Digitalization


Overview

TNGPC (Tamil Nadu Government Police Canteen) is a digital solution built to modernize the operations of police canteens across Tamil Nadu. This real-time application simplifies product management, billing, and user access, transforming manual processes into an efficient digital platform.

Tech Stack
  • Frontend: Flutter
  • Backend: Firebase / Java (optional upgrade)
  • Database: Mongodb(Cloud)
Key Features
  • Admin and User Login System
  • Inventory and Product Management
  • Instant Billing System
  • Real-time Dashboard Analytics
